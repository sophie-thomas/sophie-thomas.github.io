Accessibility Statement

This statement applies to sophie-thomas@github.io only and does not cover any of the links shared on this site. It also does not refer to any other services on the .github.io domain.

The way I have built this website means that you should be able to:

  • Zoom in up to 300% without the text spilling off the screen
  • Navigate the website using just a keyboard

I have also made the website text as simple as possible to understand.

AbilityNet has advice on making your device easier to use if you have a disability.

Contact me about an accessibility issue

I am always looking to improve the accessibility of my website to ensure it works for everyone. If you notice an accessibility issue with my website, please send me a message on LinkedIn.

Compliance

To the best of my knowledge, my website is fully compliant with the Web Content Accessibility Guidelines 2.2 to a AA standard.

It has been tested using following tools to check its compliance:

  • Lighthouse
  • Wave
  • Axe DevTools
  • ARC Toolkit
